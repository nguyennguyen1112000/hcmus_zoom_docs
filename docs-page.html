Introduction Last updated: 2022-30-05

The HCMUSID application has been built and published in the Zoom Place Market with this strategy to ease the burden on exam times as well as enhance the online supervision and equity engagements experienced by the Zoom Desktop application.

Installation

This is a guide to install HCMUSID application, including 2 parts: Installation for students and installation for administrators.

2.1 Install for student

Step 1. On Zoom App MarketPlace, find HCMUSID application. Click the Add button to install the application.

Step2 2. If the Student's Email Account has been added to the system by the administrator, it will automatically log the user into the system as a student. Otherwise, the system will notify "No user exists on the application, please contact the manager to resolve".

Note: Only when the administrator adds students to the system, the students can use the application with the Student Email account.

2.2 Install for admintrator

Step 1. On Zoom App MarketPlace, find HCMUSID application. Click the Add button to install the application.

Step 2. Agree for the application to access the Zoom account.

Step 3. In the first time installing the application, after Agreeing the access rights for the application, the user will be directly logged into the Application https://hcmus.examidentity.online depending on his role (Manager or exam supervisor)..

Note: If the App has been installed, in the next logins, users need to Sign in with their Zoom Account at the home page of the application.
